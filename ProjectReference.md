<H1>StarbucksKiosk Project</H1>

<H3>프로젝트 생성 레퍼런스</H3>

```text
[ 필요한 기능 ]
요구사항1 : 메인 메뉴판 화면
        : 장바구니 유무를 확인 후 장바구니에 제품이 있을 경우 [ORDER MENU]함께 출력
        : 메뉴 선택 시 '상세 메뉴 화면'으로 이동
        : 잘못된 번호 선택 시 예외처리
        : 프로그램 종료를 위한 번호 정의

요구사항2 : 상세 메뉴 화면
        : 메뉴 선택 시 '메뉴 확인 화면'으로 이동
        : 잘못된 번호 선택 시 예외처리
        : 프로그램 종료를 위한 번호 정의

요구사항3 : 메뉴 확인 화면
        : '확인' 선택 시 장바구니 추가 및 메인 메뉴판 화면으로 이동
        : '취소' 선택 시 이전 '상세 메뉴 화면'으로 이동
        : 잘못된 번호 선택 시 예외처리
        
요구사항4 : 장바구니 주문 화면
        : 현재 주문 대기 수 출력
        : 장바구니 목록 및 합계 금액 출력
        : '주문' 선택 시 '영업 시간'과 '은행 점검 시간'을 고려하여 예외 처리
        : '주문' 선택 시 '잔액'과 비교하여 부족할 경우 예외 처리
        : '주문' 선택 시 성공 조건일 경우 '결제 완료' 및 현재 시간 출력
        : '메뉴판' 선택 시 '메인 메뉴판 화면'으로 이동
```
